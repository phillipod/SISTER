Collection Targets:

  • Builds: {{ stats.target_per_platform_type }} accepted builds for each Platform/Type combination (e.g., PC/Space, Console/Ground).
  • Screenshots (Labels): {{ stats.target_per_label }} accepted screenshots for each overall type (Space, Ground) to ensure sufficient data for individual UI element recognition.

Special notes for screenshot labels:

  • PC Ship builds have an additional "Secondary Deflector" label not always present.
  • Ship builds have additional "Experimental Weapon" and "Hangar" labels not always present.

The "Screenshots (Labels)" target helps ensure we have enough examples of each UI element across all platforms. The "Builds" target helps ensure a good distribution of complete build examples from different platforms and types.
