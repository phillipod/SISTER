SISTER

Screenshot Interrogation System for Traits and Equipment Recognition
Build Screenshot Collection

About This Project

We're collecting Star Trek Online build screenshots to improve our build analysis tools. Your contribution will help make better tools for the STO community.

SISTER is an open-source project. Visit our GitHub repository to learn more about the project, download the latest release, or contribute.

Current release: v2025.05.31

How It Works

  1. Upload screenshots for your builds (space and/or ground)
  2. Provide your email address
  3. Select which uses you consent to
  4. Receive a detailed consent form via email
  5. Reply to confirm your consent

Screenshot Guide

For each build, please include as many of these screenshots as possible:

Space Build

  • Ship equipment and stats screen
  • Active space traits
  • Space skill tree

Ground Build

  • Character equipment and stats screen
  • Active ground traits
  • Ground skills

Privacy & Data Usage

Your privacy is important to us. Here's how we handle your data:

  • Screenshots are stored securely and only used for the purposes you consent to
  • Screenshots will be published in public GitHub repositories as specified in the consent options
  • Your email is only used for sending the consent form and will not be shared
  • You can withdraw consent at any time by creating an issue on GitHub
  • Data usage is strictly limited to improving STO build analysis tools

By submitting your screenshots, you'll receive an email with a detailed consent form. Your data will only be used after you reply to confirm your consent.

This is a community project for Star Trek Online players. Not affiliated with Cryptic Studios or Perfect World Entertainment.
